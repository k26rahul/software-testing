# Word Manipulator

A Java class with methods to perform basic string operations on a given word.

## Files

- **`WordManipulator.java`**
  - `WordManipulator(String)` - constructor, accepts the word to operate on
  - `reverse()` - returns the reversed string
  - `isPalindrome()` - checks if the word reads the same forwards and backwards
  - `countVowels()` - returns the number of vowels in the word
  - `isAnagram(String)` - checks if the word is an anagram of another word
  - `main(String[])` - entry point, runs a quick demo with a sample word

- **`WordManipulatorTest.java`**: JUnit 4 test suite for the above methods.
