# Word Manipulator Application

A Java utility class designed to perform string manipulation and analysis, including reversing words, counting vowels, checking for palindromes, and verifying anagrams.

## Architecture & Files

- **`WordManipulator.java`**: The core utility class containing methods for string operations such as reversing, palindrome checking, vowel counting, and anagram validation.
- **`WordManipulatorTest.java`**: The unit test suite using JUnit 4. It validates the correctness of the word analysis and manipulation methods. Runs natively via the VS Code "Run" button using the `util.TestHarness`.
